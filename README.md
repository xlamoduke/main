# 2x2 Card Game Engine

Чистый TypeScript-движок авторской карточной игры 2x2 без браузерного UI, React/Vue/Angular и графики. Первый этап проекта сфокусирован на стабильных правилах, тестах и консольной симуляции.

## Структура

```text
src/
  model/   # Card, Deck, Player, Team, GameState/Round/Match/Battle types
  rules/   # чистые функции сравнения карт, перебития, боя и спец-правил
  engine/  # создание раунда, добор, вклинивание, подсчёт штрафов, матч-симулятор
  cli/     # simulate.ts и demo.ts
tests/     # тесты ключевых механик правил
```

## Команды

```bash
npm install
npm test
npm run simulate
npm run demo
```

- `npm test` компилирует TypeScript и запускает тесты правил.
- `npm run simulate` прогоняет автоматические матчи и печатает статистику. Можно передать число матчей: `npm run simulate -- 10000`.
- `npm run demo` показывает заранее заданные ситуации: Shaman, no-trump, частичное перебитие, владелец банка, четыре туза, яйца и `tauntInsufficient`.

## Реализованные правила

Симулятор использует тот же движок правил, что и тесты/демо: автоматические игроки выбирают допустимые ходы, перебивают только через `canBeatSet`, добровольно сбрасывают с заданной вероятностью, получают банки и добирают карты через общие функции движка.

- 4 игрока, команды по диагонали: Player1+Player3 против Player2+Player4.
- Колода 36 карт: A, 10, K, Q, J, 9, 8, 7, 6 каждой масти.
- Очки карт: A=11, 10=10, K=4, Q=3, J=2, остальные=0.
- Обычный козырный режим и no-trump при открытии 6♠.
- Shaman: 6♠ в обычном режиме — неперебиваемая карта, может представлять любую масть и стоит 0 очков.
- Бой из ровно 4 действий без второго оборота.
- Ход 1–4 картами одной масти; Shaman может заменять масть.
- Ответ тем же количеством карт: полное перебитие последнего открытого набора или добровольный сброс.
- Перебитие через поиск полного сопоставления карт без учёта порядка; частичное перебитие запрещено.
- Банк получает команда атакующего, если исходный ход не перебили, иначе команда последнего успешно перебившего игрока; следующий бой начинает владелец банка.
- Добор от владельца банка по часовой стрелке, по одной карте за проход, только пока возможны полные круги.
- Вклинивание по 4 картам одной масти или сумме руки больше 40, с учётом Shaman как заменителя масти.
- Спец-атаки: четыре туза (+12) и четыре шестёрки (+6), включая отражение и no-trump-поведение.
- Подсчёт штрафов раунда: 0 => +6, 1–30 => +4, 31–59 => +2.
- Режим “яйца” при 60:60 с накоплением `eggDebt`.
- Матч до 12 штрафных баллов.
- UI-событие `tauntInsufficient` для игрока, перебившего четырьмя картами.
